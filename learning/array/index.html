Largest in Array

A program to find the largest element in an array.

Largest in Array

Write a program to find the largest element in an array.

Input

{5, 12, 3, 7, 20}

Output

Largest element: 20

public class LargestElement {
public static void main(String[] args) {
int[] arr = {5, 12, 3, 7, 20};
int max = arr[0];
for (int num : arr) {
if (num > max) {
max = num;
}
}
System.out.println("Largest element: " + max);
}
}
Explanation:
  1. Initialize the maximum value as the first element.
  2. Iterate through the array to compare and update the maximum.
  3. Print the largest element found.

Reverse Array

A program to reverse an array.

Reverse Array

Write a program to reverse an array.

Input

{1, 2, 3, 4, 5}

Output

Reversed array: {5, 4, 3, 2, 1}

public class ReverseArray {
public static void main(String[] args) {
int[] arr = {1, 2, 3, 4, 5};
System.out.print("Reversed array: {");
for (int i = arr.length - 1; i >= 0; i--) {
System.out.print(arr[i]);
if (i != 0) System.out.print(", ");
}
System.out.println("}");
}
}
Explanation:
  1. Start from the last index of the array and go to the first.
  2. Print each element during the loop.
  3. Use a condition to format commas properly.
Count Even & Odd

A program to count the number of even and odd numbers in an array.

Count Even & Odd

Write a program to count the number of even and odd numbers in an array.

Input

{10, 3, 8, 7, 6, 2}

Output

Even count: 4
Odd count: 2

public class CountEvenOdd {
public static void main(String[] args) {
int[] arr = {10, 3, 8, 7, 6, 2};
int even = 0, odd = 0;
for (int num : arr) {
if (num % 2 == 0)
even++;
else
odd++;
}
System.out.println("Even count: " + even);
System.out.println("Odd count: " + odd);
}
}
Explanation:
  1. Initialize counters for even and odd as 0.
  2. Loop through each element of the array.
  3. Use modulus to determine even or odd and increment counters.
  4. Print both counters.
Second Largest

A program to find the second largest element in an array.

Second Largest

Write a program to find the second largest element in an array.

Input

{12, 45, 67, 23, 89, 55}

Output

Second largest element: 67

public class SecondLargest {
public static void main(String[] args) {
int[] arr = {12, 45, 67, 23, 89, 55};
int first = Integer.MIN_VALUE, second = Integer.MIN_VALUE;
for (int num : arr) {
if (num > first) {
second = first;
first = num;
} else if (num > second && num != first) {
second = num;
}
}
System.out.println("Second largest element: " + second);
}
}
Explanation:
  1. Initialize two variables to track largest and second largest.
  2. Loop through array updating the two variables as needed.
  3. Ensure duplicates are handled and final result is printed.

Find Duplicates

A program to find duplicate elements in an array.

Find Duplicates

Write a program to find duplicate elements in an array.

Input

{3, 6, 8, 3, 2, 6, 9}

Output

Duplicate elements: 3, 6

import java.util.HashSet; public class FindDuplicates {
public static void main(String[] args) {
int[] arr = {3, 6, 8, 3, 2, 6, 9};
HashSet seen = new HashSet<>();
HashSet duplicates = new HashSet<>();
for (int num : arr) {
if (!seen.add(num)) {
duplicates.add(num);
}
}
System.out.print("Duplicate elements: ");
for (int dup : duplicates) {
System.out.print(dup + " ");
}
}
}
Explanation:
  1. Use a HashSet to track seen elements.
  2. On encountering a duplicate, add it to a duplicates set.
  3. Print the duplicate elements.
Merge Arrays

A program to merge two arrays into one.

Merge Arrays

Write a program to merge two arrays into one.

Input

{1, 3, 5} and {2, 4, 6}

Output

Merged array: {1, 3, 5, 2, 4, 6}

public class MergeArrays {
public static void main(String[] args) {
int[] arr1 = {1, 3, 5};
int[] arr2 = {2, 4, 6};
int[] merged = new int[arr1.length + arr2.length];
for (int i = 0; i < arr1.length; i++) {
merged[i] = arr1[i];
}
for (int i = 0; i < arr2.length; i++) {
merged[arr1.length + i] = arr2[i];
}
System.out.print("Merged array: ");
for (int num : merged) {
System.out.print(num + " ");
}
}
}
Explanation:
  1. Create a new array with a size equal to the sum of the two input arrays.
  2. Copy elements of the first and second arrays into the new array.
  3. Print the merged array.
Left Rotate

A program to left rotate an array by one position.

Left Rotate

Write a program to left rotate an array by one position.

Input

{10, 20, 30, 40, 50}

Output

Rotated array: {20, 30, 40, 50, 10}

public class LeftRotateByOne {
public static void main(String[] args) {
int[] arr = {10, 20, 30, 40, 50};
int first = arr[0];
for (int i = 0; i < arr.length - 1; i++) {
arr[i] = arr[i + 1];
}
arr[arr.length - 1] = first;
System.out.print("Rotated array: ");
for (int num : arr) {
System.out.print(num + " ");
}
}
}
Explanation:
  1. Store the first element of the array.
  2. Shift all other elements to the left.
  3. Place the first element at the end of the array.
