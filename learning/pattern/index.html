Hollow Square Pattern

A program to print a hollow square pattern.

Hollow Square Pattern

Write a program to print a hollow square pattern.

Input

5

Output
*****
*   *
*   *
*   *
*****
public class HollowSquare {
public static void main(String[] args) {
int n = 5; // Size of the square
for (int i = 0; i < n; i++) {
for (int j = 0; j < n; j++) {
// Print '*' for the first and last row, and first and last column
if (i == 0 || i == n - 1 || j == 0 || j == n - 1) {
System.out.print("*");
} else {
System.out.print(" ");
}
}
System.out.println();
}
}
}
Explanation:
  1. Loop through each row and column.
  2. Print '*' for the first and last row, and first and last column. Print a space for other positions.
Hollow Pyramid Pattern

A program to print a hollow pyramid pattern.

Hollow Pyramid Pattern

Write a program to print a hollow pyramid pattern.

Input

5

Output
    *
   * *
  *   *
 *     *
*********
public class HollowPyramid {
public static void main(String[] args) {
int n = 5; // Height of the pyramid
for (int i = 1; i <= n; i++) {
// Print spaces for alignment
for (int j = i; j < n; j++) {
System.out.print(" ");
}
// Print stars and spaces in the pyramid structure
for (int j = 1; j <= (2 * i - 1); j++) {
// Print '*' for the edges and the last row
if (j == 1 || j == (2 * i - 1) || i == n) {
System.out.print("*");
} else {
System.out.print(" ");
}
}
System.out.println();
}
}
}
Explanation:
  1. Use nested loops to print the correct number of spaces and stars.
  2. For each row, print spaces before the stars for alignment.
  3. Print stars at the edges of the pyramid, and spaces in between for all rows except the last one.
Hollow Diamond Pattern

A program to print a hollow diamond pattern.

Hollow Diamond Pattern

Write a program to print a hollow diamond pattern.

Input

5

Output
    *
   * *
  *   *
 *     *
*       *
 *     *
  *   *
   * *
    *
public class HollowDiamond {
public static void main(String[] args) {
int n = 5; // Number of rows for the top half of the diamond
// Top half of the diamond
for (int i = 1; i <= n; i++) {
// Print spaces before stars
for (int j = i; j < n; j++) {
System.out.print(" ");
}
// Print stars and spaces
for (int j = 1; j <= (2 * i - 1); j++) {
if (j == 1 || j == (2 * i - 1)) {
System.out.print("*");
} else {
System.out.print(" ");
}
}
System.out.println();
}
// Bottom half of the diamond
for (int i = n - 1; i >= 1; i--) {
// Print spaces before stars
for (int j = n; j > i; j--) {
System.out.print(" ");
}
// Print stars and spaces
for (int j = 1; j <= (2 * i - 1); j++) {
if (j == 1 || j == (2 * i - 1)) {
System.out.print("*");
} else {
System.out.print(" ");
}
}
System.out.println();
}
}
}
Explanation:
  1. Print the top half of the diamond using a loop for spaces and stars.
  2. In the bottom half, reverse the process to create the lower triangle.
  3. Ensure spaces are printed in between the stars for a hollow effect, except for the edges.
Butterfly Pattern

A program to print a butterfly pattern.

Butterfly Pattern

Write a program to print a butterfly pattern.

Input

4

Output
*      *
**    **
***  ***
********
********
***  ***
**    **
*      *
public class ButterflyPattern {
public static void main(String[] args) {
int n = 4; // Number of rows
// Upper half of the butterfly
for (int i = 1; i <= n; i++) {
// Print the first part of the stars
for (int j = 1; j <= i; j++) {
System.out.print("*");
}
// Print the spaces in between
for (int j = 1; j <= (2 * (n - i)); j++) {
System.out.print(" ");
}
// Print the second part of the stars
for (int j = 1; j <= i; j++) {
System.out.print("*");
}
System.out.println();
}

// Lower half of the butterfly
for (int i = n; i >= 1; i--) {
// Print the first part of the stars
for (int j = 1; j <= i; j++) {
System.out.print("*");
}
// Print the spaces in between
for (int j = 1; j <= (2 * (n - i)); j++) {
System.out.print(" ");
}
// Print the second part of the stars
for (int j = 1; j <= i; j++) {
System.out.print("*");
}
System.out.println();
}
}
}
Explanation:
  1. In the upper half, print stars on both sides and spaces in between.
  2. The number of spaces decreases as the number of stars increases for each row.
  3. The lower half is a mirror image of the upper half, which is printed after the upper half is completed.

Right Pascal's Triangle

Write a program to print a right Pascal's triangle.

Input

5

Output
*
**
***
****
*****
****
***
**

public class RightPascalsTriangle {
public static void main(String[] args) {
int n = 5; // Number of rows
// Upper half of Pascal's Triangle
for (int i = 1; i <= n; i++) {
for (int j = 1; j <= i; j++) {
System.out.print("*");
}
System.out.println();
}
// Lower half of Pascal's Triangle
for (int i = n - 1; i >= 1; i--) {
for (int j = 1; j <= i; j++) {
System.out.print("*");
}
System.out.println();
}
}
}
Explanation:
  1. In the upper half, print an increasing number of stars for each row.
  2. In the lower half, print a decreasing number of stars for each row, starting from the row before the peak.

Hourglass Pattern

A program to print an hourglass pattern.

Hourglass Pattern

Write a program to print an hourglass pattern.

Input

5

Output
*********
 *******
  *****
   ***
    *
   ***
  *****
 *******

public class HourglassPattern {
public static void main(String[] args) {
int n = 5; // Number of rows
// Upper half of the hourglass
for (int i = 0; i < n; i++) {
// Print leading spaces
for (int j = 0; j < i; j++) {
System.out.print(" ");
}
// Print stars
for (int j = 0; j < (2 * (n - i) - 1); j++) {
System.out.print("*");
}
System.out.println();
}
// Lower half of the hourglass
for (int i = 1; i < n; i++) {
// Print leading spaces
for (int j = 0; j < n - i - 1; j++) {
System.out.print(" ");
}
// Print stars
for (int j = 0; j < (2 * (i + 1) - 1); j++) {
System.out.print("*");
}
System.out.println();
}
}
}
Explanation:
  1. In the upper half, print decreasing stars with leading spaces increasing in each row.
  2. In the lower half, print increasing stars with leading spaces decreasing in each row.
Alphabet Triangle Pattern

A program to print an alphabet triangle pattern.

Alphabet Triangle Pattern

Write a program to print an alphabet triangle pattern.

Input

5

Output
A
AB
ABC
ABCD
ABCDE
public class AlphabetTrianglePattern {
public static void main(String[] args) {
int n = 5; // Number of rows
// Loop for each row
for (int i = 1; i <= n; i++) {
// Loop to print the alphabet letters
for (char j = 'A'; j < 'A' + i; j++) {
System.out.print(j);
}
System.out.println();
}
}
}
Explanation:
  1. In each row, print the letters starting from 'A' up to the current row number.
  2. The number of letters in each row increases as the rows increase.
Zig-Zag Pattern

A program to print a zig-zag pattern.

Zig-Zag Pattern

Write a program to print a zig-zag pattern.

Input

7

Output
  *   *   *  
 * * * * * * 
*   *   *   *
public class ZigZagPattern {
public static void main(String[] args) {
int n = 7; // Number of columns
// Loop to print the upper part of the zig-zag pattern
for (int i = 1; i <= 3; i++) {
for (int j = 1; j <= n; j++) {
if (j % 2 != 0) {
System.out.print("* ");
} else {
System.out.print(" ");
}
}
System.out.println();
}

// Loop to print the bottom part of the zig-zag pattern
for (int i = 1; i <= 3; i++) {
for (int j = 1; j <= n; j++) {
if (j % 2 == 0) {
System.out.print("* ");
} else {
System.out.print(" ");
}
}
System.out.println();
}
}
}
Explanation:
  1. For each row, we alternate between printing stars and spaces.
  2. The pattern forms a zig-zag structure by alternating the direction of stars in subsequent rows.
Binary Triangle Pattern

A program to print a binary number triangle pattern.

Binary Triangle Pattern

Write a program to print a binary number triangle pattern.

Input

5

Output
1
01
101
0101
10101
public class BinaryTrianglePattern {
public static void main(String[] args) {
int n = 5; // Number of rows
for (int i = 1; i <= n; i++) {
for (int j = 1; j <= i; j++) {
if ((i + j) % 2 == 0) {
System.out.print("1");
} else {
System.out.print("0");
}
}
System.out.println();
}
}
}
Explanation:
  1. For each row, print the binary digits alternately starting from 1.
  2. Use the condition `(i + j) % 2 == 0` to decide whether to print '1' or '0'.
