Convert String to Character Array

A program to convert a string to a character array.

Convert String to Character Array

Write a program to convert a string to a character array.

Input

"Hello"

Output

Character Array: ['H', 'e', 'l', 'l', 'o']

public class ConvertStringToCharArray {
public static void main(String[] args) {
String str = "Hello";
char[] charArray = str.toCharArray();
System.out.print("Character Array: [");
for (int i = 0; i < charArray.length; i++) {
System.out.print("'" + charArray[i] + "'");
if (i < charArray.length - 1) {
System.out.print(", ");
}
}
System.out.println("]");
}
}
Explanation:
  1. Use the toCharArray() method to convert a string into a character array.
  2. Print each character in the array to display the character array format.
Convert Integer to Binary, Octal, and Hexadecimal

A program to convert an integer to binary, octal, and hexadecimal.

Convert Integer to Binary, Octal, and Hexadecimal

Write a program to convert an integer to binary, octal, and hexadecimal.

Input

25

Output

Binary: 11001

Octal: 31

Hexadecimal: 19

public class ConvertInteger {
public static void main(String[] args) {
int number = 25;
// Convert to binary, octal, and hexadecimal
String binary = Integer.toBinaryString(number);
String octal = Integer.toOctalString(number);
String hexadecimal = Integer.toHexString(number).toUpperCase();
System.out.println("Binary: " + binary);
System.out.println("Octal: " + octal);
System.out.println("Hexadecimal: " + hexadecimal);
}
}
Explanation:
  1. Use the Integer.toBinaryString() method to convert the integer to binary.
  2. Use the Integer.toOctalString() method to convert the integer to octal.
  3. Use the Integer.toHexString() method to convert the integer to hexadecimal.
  4. Convert the hexadecimal to uppercase for proper formatting.
Convert Binary to Decimal

A program to convert a binary number to a decimal.

Convert Binary to Decimal

Write a program to convert a binary number to a decimal.

Input

"1010"

Output

Decimal: 10

public class BinaryToDecimal {
public static void main(String[] args) {
String binary = "1010"; // Binary number as a string
int decimal = Integer.parseInt(binary, 2); // Convert binary to decimal using base 2
System.out.println("Decimal: " + decimal);
}
}
Explanation:
  1. The Integer.parseInt() method is used to convert a string to an integer.
  2. The second argument of parseInt() specifies the base of the number (2 for binary).
  3. The result is the decimal equivalent of the binary number.

Convert Fahrenheit to Celsius

A program to convert temperature from Fahrenheit to Celsius.

Convert Fahrenheit to Celsius

Write a program to convert temperature from Fahrenheit to Celsius.

Formula: C = (F - 32) * 5/9

Input

100 (Fahrenheit)

Output

Celsius: 37.78

public class FahrenheitToCelsius {
public static void main(String[] args) {
double fahrenheit = 100; // Fahrenheit temperature
double celsius = (fahrenheit - 32) * 5 / 9; // Convert to Celsius
System.out.println("Celsius: " + String.format("%.2f", celsius));
}
}
Explanation:
  1. The formula C = (F - 32) * 5/9 is used to convert Fahrenheit to Celsius.
  2. The result is displayed with two decimal places using String.format("%.2f", celsius).
